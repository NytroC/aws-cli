# aws-cli

## Installation

* Install ruby
* Pull Repo

then run commands:  
* gem install bundler
* bundle install


## Running  

mode options:  
`put_song`  
`put_album`  
`put_artist`  

Example:  
`ruby aws-cli.rb put_song artist="Miley Cyrus" album="Bangers"  name="Younger Now" location="path/to/file"`
